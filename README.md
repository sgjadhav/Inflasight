# Inflasight

Inflasight is a data-driven project that analyzes official CPI inflation data
and computes **personalized inflation** based on individual spending patterns.

## Key Features
- Uses official CPI group-wise inflation data (MOSPI)
- Calculates personal inflation using user-defined spending weights
- Supports time-series analysis and forecasting
- ML-based user segmentation and inflation prediction (planned)

## Tech Stack
- Python (ETL, backend, ML)
- PostgreSQL (database)
- FastAPI (backend API)
- React (frontend)
- Git & GitHub (version control)
